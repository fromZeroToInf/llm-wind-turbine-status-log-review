# AI assisted status log checker
LLM assisted status log reviewer to find relevant messages for anomaly detections and to prioritize relevant detections.

This project is a follow-up of my project [Explorative-AD-in-Wind-Turbine-SCADA-Data-Using-an-AE](https://github.com/fromZeroToInf/Explorative-AD-in-Wind-Turbine-SCADA-Data-Using-an-AE). A big problem or a important pain point was in that project that i had to manually check the detections visually and to search for further informations in the status logs (which is a common workflow), due to limited available time, i could not pursue a comprehensive investigation of the detections.

This project aims to target to filter undocumented detection events.
